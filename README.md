# Food Ordering System

## Overview
A comprehensive food ordering web application built using Java, JSP, Servlets, and MySQL.

## Features
- User login and registration
- Browse menu items
- Place and track orders
- Admin dashboard to manage menu items

## Tech Stack
- Backend: Core Java, JSP, Servlets
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- Tools: Eclipse, Tomcat


## How to Run
1. Import project in Eclipse
2. Configure Tomcat server
3. Run on localhost
4. Open browser and navigate to `http://localhost:8080/FoodOrderingSystem`
